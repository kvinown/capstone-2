# capstone-2
 aplikasi web untuk melakukan pengajuan beasiswa internal dengan studi kasus di Fakultas Teknologi Informasi Universitas Kristen Maranatha. Terdapat 4 role pengguna dalam aplikasi ini yaitu administrator, mahasiswa, pengguna program studi (ketua program studi), dan pengguna fakultas (dekan, wakil dekan). Berikut adalah hal yang dapat dilakukan oleh masing-masing pengguna
